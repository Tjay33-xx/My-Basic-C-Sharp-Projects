Car Insurance MVC Project

A web application built using ASP.NET Core MVC and Entity Framework that calculates insurance quotes based on user input.

### Features:
- MVC architecture (Model-View-Controller)
- Entity Framework Code-First database
- Dynamic quote calculation system
- CRUD operations for insurees
- Admin view showing all quotes

### Technologies Used:
- C#
- ASP.NET Core MVC (.NET 8)
- Entity Framework Core
- SQL Server
- Razor Views
